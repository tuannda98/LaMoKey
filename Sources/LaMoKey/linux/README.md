# LaMoKey
### Open source Vietnamese Input App for Linux

LaMoKey Linux đang trong giai đoạn phát triển nhưng đã hỗ trợ kiểm tra cập nhật phiên bản.

## Kiểm tra cập nhật phiên bản

Sau khi build project, chạy:

```bash
./lamokey_update_check
```

Nếu có phiên bản mới hơn, công cụ sẽ thông báo và trả về mã lỗi `1`.

## Build nhanh

```bash
mkdir -p build && cd build
cmake ..
cmake --build .
```
